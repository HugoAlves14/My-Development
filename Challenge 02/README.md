Challenge: Complete Grading System (with functions)

Create a Python program that:

Data input:

Ask for the student's name
Ask for 3 grades (decimal values)

System rules:

Create a function that receives the 3 grades
Calculate:
Average
Highest grade
Lowest grade

Student status:

Average greater than or equal to 7 → Passed
Average between 5 and 6.9 → Remedial
Average less than 5 → Failed

Output:

Show the student's name
Show the average with two decimal places
Show the highest grade
Show the lowest grade
Show the status

Example of expected output:

Name: João
Average: 6.50
Highest grade: 8.0
Lowest grade: 5.0
Status: Remedial

Extra challenge (Optional):

Do not allow grades lower than 0 or higher than 10.
Use validation with a while loop to ensure correct values.
Separate the code into multiple functions (one for calculation, another for display).
